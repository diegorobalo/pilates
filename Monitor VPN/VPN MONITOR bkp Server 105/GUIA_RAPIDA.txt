│ 📱 LAS 7 PESTAÑAS EXPLICADAS                                                   │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  1️⃣  CONECTADOS          ➜  ¿Quién está usando VPN AHORA?                    │
│      • Nombres de usuarios conectados en este momento                         │
│      • Ubicación geográfica (Ciudad/País)                                     │
│      • Hora de conexión actual                                                │
│      • Datos descargados/subidos                                              │
│      ✓ USO: Control de presencia en tiempo real                               │
│                                                                                │
│  2️⃣  TABLA DETALLE       ➜  Vista más técnica de conectados ahora             │
│      • IP asignada a cada usuario                                             │
│      • Identificador del dispositivo (PC, móvil, etc)                         │
│      • Consumo de ancho de banda detallado                                    │
│      ✓ USO: Análisis técnico de conexiones                                    │
│                                                                                │
│  3️⃣  HISTORIAL          ➜  ¿Qué pasó en los últimos días?                    │
│      • Cada sesión completada (inicio y fin)                                  │
│      • Horas de trabajo por sesión                                            │
│      • Datos transferidos por sesión                                          │
│      ✓ USO: Revisar historial de conexiones                                   │
│                                                                                │
│  4️⃣  AUDITORÍA          ➜  CONTROL DE ASISTENCIA                             │
│      • Detecta usuarios sin conectar 3+ días                                  │
│      • Identifica ausentes del trabajo                                        │
│      • Muestra patrones de ausentismo                                         │
│      • Buscar por nombre para ver detalles                                    │
│      ✓ USO: Justificaciones de ausencias y control diario                     │
│                                                                                │
│  5️⃣  INFORME USUARIO    ➜  REPORTE INDIVIDUAL DE PRODUCTIVIDAD               │
│      • Gráfico de horas por día (últimos 30 días)                             │
│      • Distribución por día de semana                                         │
│      • Total de horas trabajadas                                              │
│      • Tendencias y promedios                                                 │
│      ✓ USO: Evaluar desempeño individual de empleados                         │
│                                                                                │
│  6️⃣  RRHH / GESTIÓN     ➜  ANÁLISIS GLOBAL DE RECURSOS HUMANOS                │
│      • Top 10 usuarios más productivos                                        │
│      • Resumen de horas por sector/departamento                               │
│      • Mapa de calor de actividad (qué días hay más trabajo)                  │
│      • Detección de ausencias prolongadas                                     │
│      • ASIGNAR usuarios a departamentos                                       │
│      ✓ USO: Análisis de productividad por sector y decisiones RRHH            │
│                                                                                │
│  7️⃣  IT / SEGURIDAD     ➜  MONITOREO TÉCNICO Y DE SEGURIDAD                  │
│      • Horas pico de conexión (cuándo más se usa)                             │
│      • Ubicaciones de conexión (países/ciudades)                              │
│      • Detección de anomalías                                                 │
│      ✓ USO: IT y Seguridad (no es para RRHH)                                  │
│                                                                                │

│  TAREA: "Necesito saber quién faltó hoy"                                      │
│  ─────────────────────────────────────────                                    │

│  2. Busca el nombre del empleado                                              │

│  3. Haz clic: [Generar]                                                       │
│  4. Mira los gráficos y el promedio de horas/día                              │
│     • 5+ horas/día = Productivo ✓                                             │
│     • 3-5 horas/día = Normal ✓                                                │

│     • Recomendaciones                                                         │

┌────────────────────────────────────────────────────────────────────────────────┐
│ 📊 CÓMO INTERPRETAR NÚMEROS                                                    │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  HORAS:                                                                       │
│  • 7-8 horas/día = Jornada laboral completa ✓                                 │
│  • 4-6 horas/día = Medio día de trabajo ⚠️                                     │
│  • 0-3 horas/día = Muy poco trabajo ✗                                         │
│                                                                                │
│  BYTES / MB / GB:                                                              │
│  • MB = Megabytes (1000 KB) - archivos pequeños                               │
│  • GB = Gigabytes (1000 MB) - archivos grandes, videos                        │
│  • Normal: 10-30 MB por hora de trabajo                                       │
│  • Anormal: > 50 MB por hora (posible descarga excesiva)                      │
│                                                                                │
│  DÍAS ACTIVOS:                                                                │
│  • 20-22 días en un mes = Normal (descuenta fines de semana)                  │
│  • 25-30 días = Muy productivo                                                │
│  • < 15 días = Revisar asistencia                                             │
│                                                                                │

┌────────────────────────────────────────────────────────────────────────────────┐
│ 🚨 SEÑALES DE ALERTA                                                           │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  ⚠️  USUARIO CON 0 HORAS HOY                                                   │
│      → Significa: No se conectó a VPN hoy                                     │
│      → Acción: Revisar si está de vacaciones/licencia/ausente                 │
│                                                                                │
│  ⚠️  CONEXIONES A OTRAS CIUDADES/PAÍSES                                        │
│      → Significa: El usuario se conectó desde otro lugar                      │
│      → Acción: Si no es viaje de negocios, investigar                         │
│                                                                                │
│  ⚠️  HORAS MUY BAJAS (< 2 horas/día)                                            │
│      → Significa: Usuario no trabaja mucho desde el VPN                       │
│      → Acción: Entrevistar para entender por qué                              │
│                                                                                │
│  ⚠️  DATOS MUY ALTOS (> 100 MB/hora)                                            │
│      → Significa: Usuario descarga muchos datos                               │
│      → Acción: Revisar si es trabajo legítimo o problema técnico              │
│                                                                                │
│  ⚠️  CONEXIÓN A LAS 3 AM                                                       │
│      → Significa: Usuario trabajando fuera de horario                         │
│      → Acción: Revisar si es autorizado o si hace horas extra no pagadas      │
│                                                                                │

│ ❓ PREGUNTAS FRECUENTES                                                         │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  P: ¿El VPN Monitor sabe QUÉ está haciendo cada usuario?                      │
│  R: No. Solo ve cuándo se conecta/desconecta. No ve documentos,               │
│     emails, ni lo que escriben. Es un monitor de PRESENCIA, no de CONTENIDO.  │
│                                                                                │
│  ────────────────────────────────────────────────────────────────────────────  │
│                                                                                │
│  P: ¿Es legal usar este monitor?                                              │
│  R: Sí, si se informa a los empleados. Es monitoreo de conectividad,          │
│     no de contenido. Consultar con abogados de la empresa.                    │
│                                                                                │
│  ────────────────────────────────────────────────────────────────────────────  │
│                                                                                │
│  P: ¿Se guarda el historial de años anteriores?                               │
│  R: Sí, todo se guarda en la base de datos. Puedes cambiar el                 │
│     selector de "Días" a 60, 90 o 365 para ver más atrás.                     │
│                                                                                │
│  ────────────────────────────────────────────────────────────────────────────  │
│                                                                                │
│  P: ¿Qué pasa si un usuario se desconecta accidentalmente?                    │
│  R: Se registra como una sesión finalizada. Si se conecta de nuevo            │
│     en 5 minutos, aparece como otra sesión. Es normal.                        │
│                                                                                │
│  ────────────────────────────────────────────────────────────────────────────  │
│                                                                                │
│  P: ¿Por qué hay días con 0 conexiones pero el usuario dice que trabajó?      │
│  R: Probablemente trabajó sin VPN (red local, wifi otro sitio,                │
│     dispositivo diferente). Hay que revisar caso a caso.                      │
